# 📝 To-Do List App

A responsive and user-friendly **To-Do List** web application built using **HTML, CSS, and JavaScript**. This project helps users organize their daily tasks by allowing them to add, edit, complete, and delete tasks. All tasks are stored in the browser using **Local Storage**, so they remain available even after refreshing or closing the browser.

---

## 🚀 Features

* ➕ Add new tasks
* ✏️ Edit existing tasks
* ✅ Mark tasks as completed
* 🗑️ Delete tasks
* 💾 Automatic Local Storage support
* ⌨️ Add tasks by pressing the **Enter** key
* 📱 Fully responsive design
* 🎨 Modern and clean user interface

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the application
* **CSS3** – Styling and responsive design
* **JavaScript (ES6)** – Application logic and DOM manipulation
* **Local Storage API** – Persistent task storage

## ⚙️ How to Run the Project

1. Download or clone this repository.
2. Open the project folder.
3. Double-click **index.html** or open it in any modern web browser.
4. Start adding and managing your tasks.

---

## 💡 How It Works

* Enter a task in the input field.
* Click the **Add** button or press **Enter**.
* Click the **✔** icon to mark a task as completed.
* Click the **✏️** icon to edit a task.
* Click the **🗑️** icon to delete a task.
* All changes are automatically saved using Local Storage.
